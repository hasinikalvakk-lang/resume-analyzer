# Resume Analyzer Web Application 🚀

This is a modern Resume Analyzer Web Application built using Python and Flask.

## Features

- Resume PDF Upload
- ATS Score Analysis
- Skill Detection
- Missing Skills Section
- Resume Text Extraction
- Professional Dashboard UI

## Tech Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- pdfplumber

## Live Demo

[click here](https://resume-analyzer-two-plum.vercel.app/)

## How to Run

Install requirements:

```bash
pip install flask
pip install pdfplumber
pip install python-docx
```

Run the app:

```bash
python app.py
```

Open in browser:

```bash
http://127.0.0.1:5000
```
